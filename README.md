## 📌 Project Overview

This project involves the **design, analysis, and investigation** of a reliable digital communication system that transmits data over an **AWGN (Additive White Gaussian Noise)** wireless channel under strict constraints of bandwidth, power, and Bit Error Rate (BER).

### System Specifications
- **Channel Type:** AWGN
- **Bandwidth:** 10 MHz
- **Transmitter Power:** 5 Watts
- **Target BER:** < 10⁻⁶
- **Goal:** Maximize data rate while maintaining the required BER

---

## 🛠️ Key Tasks Completed

1. **Modulation Scheme Selection**  
   Comparative analysis of BPSK, QPSK, 16-QAM (and others) based on spectral efficiency and BER performance.

2. **Error Control Coding**  
   Implementation of Convolutional Coding to achieve the target BER with coding gain analysis.

3. **System Design & Optimization**  
   Selection of best modulation + coding combination under power and bandwidth constraints.

4. **Performance Evaluation**  
   BER vs SNR curves, coding gain calculation, data rate calculation, and trade-off analysis.
